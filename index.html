Current Projects

I regularly work on a number of projects to practice a variety of web development skills. Here is an assortment of what I'm currently working on.

Sick Fits

Sick Fits

In an effort to stay up-to-date with the latest full-stack web development technologies and best-practices, I am currently working through a course to build an online store platform built with React, Apollo Client, GraphQL Yoga, and Prisma GraphQL Database Interface.

I chose this course to round out my full-stack web development skills by exposing me to more in-depth javascript testing, permission management, JWT authentication, and credit card integration.

Landlorder

Landlorder

To solidify my understanding of React and GraphQL, I am building a landlord management application in parallel to the React/GraphQL course described above.

This application will allow landlords to list, and manage their properties, communicate with tenants, employees, and contractors, log invoices, and manage calendars and todo lists.

Once these features are complete, I plan to add some social features that will allow tenants at the same property to communicate with eachother.

Hummingbird Hill

Hummingbird Hill

My parents recently opened a bed and breakfast, and are hoping to move away from using third party sites like Airbnb to book guests. I am currently building them a simple website.

Once the design is finalized, I plan to build the final version in React to create a snappy performance for prospetive guests, and will eventually integrate online booking via a separate web application I am also building.

Vivlia

Vivlia

In addition to building their website, I am also building a booking management application for my parent's bed and breakfast. This applcation will allow them to manage bookings, communicate with customers, and manage to-do lists related to running the business.

This application uses the MERN stack. MongoDB, Express, React, and NodeJS, and will interface via API with a front-end calendar that can be embeded on a website. This is a live demo of the calendar.
